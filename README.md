# Elememo

Simple Note App for Mac.

### Dev

Using '''mainWindow.loadURL(`http://localhost:4200`);''' in ./index.js file.

Running ng server
~~~
$ npm start
~~~

Start electron
~~~
$ npm run electron
~~~


### Product

Using '''mainWindow.loadURL(`file://${__dirname}/public/index.html`);''' in ./index.js file.

Generate js files
~~~
$ npm run build:prod
~~~

Start electron
~~~
$ npm run electron
~~~
